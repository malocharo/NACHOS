Detailed Description

Routines to manage an open Nachos file.

As in UNIX, a file must be open before we can read or write to it. Once we're all done, we can close it (in Nachos, by deleting the OpenFile data structure).

Also as in UNIX, for convenience, we keep the file header in memory while the file is open.
