Detailed Description

template<class Priority>
class ListElement< Priority >

definition of a "list element"

The class defines a "list element" which is used to keep track of one item on a list. It is equivalent to a LISP cell, with a "car" ("next") pointing to the next element on the list, and a "cdr" ("item") pointing to the item on the list.

Internal data structures kept public so that List operations can access them directly.
