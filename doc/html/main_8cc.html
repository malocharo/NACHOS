Detailed Description

Bootstrap code to initialize the operating system kernel.

Usage: nachos -d <debugflags> -s -x <nachos file>=""> -z -f <configfile>

-d causes certain debugging messages to be printed (cf. utility.h) -s causes user programs to be executed in single-step mode -z prints the copyright message -f <configfile> gives the name of a configuration file for Nachos -x runs a user program
