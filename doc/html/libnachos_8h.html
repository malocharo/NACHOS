Detailed Description

Function structures for programs.

Libnachos proposes several 'libc-like' functions for: Input-Output operations, String operations, Memory operations, System calls are defined in kernel/syscalls.h

Nachos-libc functions are prefixed by 'n' to avoid any confusion with standard libc functions.
