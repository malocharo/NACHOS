Detailed Description

Routines implementing an ACIA device.

Provide reading methodes on ACIA's state registers and on input/output data register. Two methods provide the reception/emision of a character from/to a remote machine. Methods are also provided to set/get the ACIA working mode (Busy Waiting or Interrupt).
